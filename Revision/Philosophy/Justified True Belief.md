# Text Elements
Justified True Belief ^3KujC9RS

Jusification ^L7jL0rWT

A claim without the ability to justify is simply 
an unsubstantiated assertion. An unjustified
true belief is not knowledge (e.g. racist juror).
Justification criteria are required to distinguish
knowledge from a guess/estimate/luck ^0n7EmxrU

Belief ^lnwaAIsH

You cannot know something without also believing it.
There is an argument to say that knowledge and belief
are incompatible, as knowledge is fallible and belief is fallible.
However, this is based upon acquaintance knowledge, which is
unconscious, and belief operates at a more conscious level ^j2Nriimi

Truth ^YtUKYc5Q

For something to be classed as knowledge, it msut be true.
If a proposition is false, it is merely a belief.

The understanding of truth is contested, and two main theories
prevail:

1) The correspondence between truth and objective reality constitutes
truth

2) Coherence: a belief is true if it is part of a 'web of beliefs
 held by society to be true. This allows truth to be objectively
false, asnd relies upon other beliefs. It allows those in other time
periods to have knowledge  ^aKpDCthR
